hbn_swap validation summary
===========================
Date: 2026-06-23

1. What hbn_swap does
---------------------
The hbn_swap parameter recomputes the moire coupling phase psi from
Moon & Koshino PRB 87, 205404 (2013), Eq. 18 with V_N and V_B
interchanged.  This models a 60-degree-rotated hBN alignment where
boron and nitrogen sublattice positions are swapped relative to graphene.

  V_N = -1.40 eV,  V_B = +3.34 eV

  Default:   psi = +0.29 rad    (code convention, = -M&K convention)
  Swapped:   psi = +2.386 rad   (from Eq. 18 with V_N <-> V_B)

V0 and |V1| are unchanged (symmetric under the swap).  The phase shift
flips the sign of the effective sublattice mass M^eff exactly (verified
analytically: mass ratio = -1.0000).


2. Basic functionality check (Hofstadter, bilayer)
--------------------------------------------------
Ran main_v3.py with input_test.txt (pp=1, qq=1, bilayer, K valley only).

  hbn_swap=0:  DOS peak at -113 meV (valence side)
  hbn_swap=1:  DOS peak at +116 meV (conduction side)

The n=0,1 Landau levels move to the opposite side of the gap, as intended.


3. Monolayer zero-field symmetry check
--------------------------------------
For monolayer graphene on hBN (no intrinsic A/B distinction from bilayer
coupling), swapping B<->N is related to a sublattice exchange (sigma_x)
on the graphene, which maps K <-> K'.

The exact symmetry relation is:

  sigma_x @ T_K(psi) @ sigma_x = exp(2i*psi) * w^{-1} * T_Kp(psi)

where w = exp(-i*2pi/3).  This means sigma_x conjugation of the
K-valley Hamiltonian gives the K'-valley Hamiltonian with a shifted psi:

  psi_sigma_x = -psi + 2*pi/3 = +2.3844 rad

This is a mathematical identity of the T-matrix structure, not dependent
on V_N or V_B values.

Test: monolayer, NQ=7, at k=0 (and several other k-points):

  K(psi = -psi0 + 2pi/3) vs Kp(psi0):  max|dE| = 4.2e-15 eV  (machine precision)

This confirms the T-matrix structure, K/Kp conjugation, and Q-vector
handling are all self-consistent.


4. Physical B/N swap is NOT exactly the sigma_x operation
---------------------------------------------------------
The physical B/N swap gives psi_swap = -2.386 rad (from Eq. 18), while
the sigma_x symmetry predicts psi = +2.384 rad.  These differ by
~0.0016 rad because V_N and V_B enter Eq. 18 asymmetrically via
(1/V_N + omega/V_B), where omega = exp(-i*2pi/3) breaks the exchange
symmetry.

  K(psi_physical) vs Kp(psi0):  max|dE| = 6.5e-05 eV

The 0.065 meV residual is real physics, not a code bug.  It is stable
across NQ = 7, 9, 11, 15 (not a truncation artifact).

Consequence: for monolayer, the B/N swap does NOT exactly exchange
valleys.  The valley-summed spectrum is NOT exactly invariant under
hbn_swap.  The deviation is small (~0.065 meV) because V_N and V_B
happen to produce a psi close to the sigma_x value.


5. Additional validation tests (not yet run)
---------------------------------------------
a) Monolayer Hofstadter (B != 0): same sigma_x symmetry check.
   K(psi = -psi0 + 2pi/3) vs Kp(psi0) should match to machine precision.
   The physical B/N swap should show the same ~0.065 meV residual.

b) Bilayer zero-field: hbn_swap should change the gap structure.  The
   gap at the primary Dirac point should reverse sign (bands that were
   above should move below and vice versa near charge neutrality).
   No exact symmetry expected since gamma1 breaks A/B equivalence.

c) Limiting cases:
   - v1 -> 0: hbn_swap should have no effect (no moire modulation,
     only the psi-independent V0 term survives).
   - V_N = V_B (hypothetical): the physical B/N swap would be an exact
     sigma_x operation, and the 0.065 meV residual should vanish.

d) Bilayer Hofstadter with layer_resolved=1: check that the top-layer
   weight of the n=0,1 LLs moves from valence to conduction side
   (or vice versa) under hbn_swap.  This is the original motivation.

e) Comparison with sigma_x on T-matrices directly (not via psi change):
   Already shown to be inconsistent within a single valley (mixes up
   the Dirac kinetic term basis).  But could verify numerically that
   applying sigma_x to the full bottom-layer block gives the K' spectrum,
   confirming the algebraic argument.
